## 環境変数要件

### 現在必要（設定済み想定）
```
NEXT_PUBLIC_FIREBASE_*
FIREBASE_ADMIN_*
ANTHROPIC_API_KEY
```

### 追加設定が必要
```
# Google Sheets連携
GOOGLE_SHEETS_API_KEY
GOOGLE_SHEETS_CLIENT_EMAIL
GOOGLE_SHEETS_PRIVATE_KEY

# LINE WORKS Admin API
LINEWORKS_CLIENT_ID
LINEWORKS_CLIENT_SECRET
LINEWORKS_SERVICE_ACCOUNT

# 音声文字起こし
OPENAI_API_KEY  # Whisper用

# GMOあおぞらネット銀行（本番用）
GMO_BANK_API_URL
GMO_BANK_CLIENT_ID
GMO_BANK_CLIENT_SECRET
GMO_BANK_ACCOUNT_ID
```
